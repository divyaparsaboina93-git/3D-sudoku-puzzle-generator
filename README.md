# 🧩 3D Sudoku Puzzle Generator

A visually interactive 3D-style Sudoku Puzzle Generator built using **HTML, CSS, and JavaScript**.  
It generates Sudoku puzzles with different difficulty levels and provides a smooth and engaging solving experience.

---

## ✨ Features
- 🎮 Interactive Sudoku grid
- 🔢 Auto puzzle generation
- 📊 Multiple difficulty levels (Basic, Intermediate, Advanced)
- 🎨 Clean and responsive UI with 3D-style design
- 🔄 Easy reset and restart functionality

---

## 🛠️ Tech Stack
- HTML
- CSS
- JavaScript
